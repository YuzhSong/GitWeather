# GitWeather

GitWeather is a playful developer dashboard that turns your Git activity into daily coding weather, mood, and productivity insights.

## What It Does

- Reads a local Git repository path.
- Looks at commits from the last 7 days.
- Calculates additions, deletions, changed files, TODO/FIXME count, merge activity, and commit-message stress signals.
- Generates a daily coding weather report.
- Gives branches personality labels, because `main` absolutely has old-dad energy.

## Quick Start

```bash
npm run dev
```

Open:

```text
http://127.0.0.1:4177
```

The API also runs on the same local server:

```text
http://127.0.0.1:4177
```

## MVP Features

- Code weather card: sunny, cloudy, rainy, stormy, or foggy.
- Productivity, debug pressure, and stability gauges.
- Code temperature, commit humidity, bug wind speed, and developer comfort.
- Branch personality panel.
- Recent commit timeline.
- Local-only analysis. No source code is uploaded.
